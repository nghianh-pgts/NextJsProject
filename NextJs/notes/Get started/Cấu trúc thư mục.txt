
#Thư mục .next
- là thư mục được tự động tạo ra khi tạo dự án, chứa code server

#Thư mục public
- Lưu trữ các tệp tĩnh như hình ảnh, favicon, tài liệu PDF, hoặc tệp robots.txt.

#Thư mục src
- chứa code chính
- Tổ chức mã nguồn dự án. Nếu sử dụng thư mục này, cần di chuyển pages, components, styles, và các thư mục liên quan vào đây

#file component.json
- chứa cấu hình của dự án
- để cho shadcn ui terminal(shadcn ui CLI)  hiểu được dự án đang setting gì
